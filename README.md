# Excel Macros

This repository contains the following Excel .xlsm or .xlam files.

## MakeAppScreenshots.xlsm

### Purpose

To create screenshots for distributing an iOS or Android app.

### Required

4 .jpg or .png files of screenshots saved while running the app on an actual device.

### Method

- Open MakeAppScreenshots.xlsm file in Excel.
- Edit Inp sheet's range B1 to B11.
- Run MakeAppScreenshots macro (from Developer menu's Macros submenu).
- Obtain the output images in Out folder within the folder of input images.
